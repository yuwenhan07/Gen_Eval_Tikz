\section*{Monoid $M = \langle a, b \mid ba = aa \rangle$: Upho Lattice with Core $M_2$}

The following diagram represents the upho lattice for the monoid $M = \langle a, b \mid ba = aa \rangle$, corresponding to the case $r=2$ of \cref{thm:rank_two}. The core of this lattice is $M_2$.

\begin{figure}[h]
\centering
\begin{tikzpicture}[
  level distance=1.5cm, % Distance between levels
  sibling distance=1.5cm, % Horizontal distance between siblings
  every node/.style={align=center, font=\small}, % Style for all nodes
  edge from parent/.style={-}, % Style for edges
  grow'=up % Grow tree upwards
]

% Define the root node
\node {}
  child {
    node {1}
    child { node {$a$} }
    child { node {$b$} }
  }
  child {
    node {$ab$}
    child { node {$aab$} }
    child { node {$abb$} }
  }
  child {
    node {$aa$}
    child { node {$aaa$} }
    child { node {$bbb$} }
  }
  child {
    node {$bb$}
  };
\end{tikzpicture}
\caption{Upho lattice for $M = \langle a, b \mid ba = aa \rangle$ with core $M_2$.}
\label{fig:upho_lattice}
\end{figure}
